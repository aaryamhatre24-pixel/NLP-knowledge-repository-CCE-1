Section C: Structured NLP Workflow Charts

Aarya Yugant Mhatre & Shraddha Sandip Salvi

Stage 01: Text Preprocessing Workflow

Sentence Segmentation Splits long text blocks into distinct sentences.
→
Tokenization Breaks sentences into words or subwords.
→
Noise Removal Strips out HTML tags, markup, and structural metadata.
→
Case Normalization Converts all characters uniformly to lower-case.
→
Stop-word Removal Filters out common uninformative structural words.
→
Lemmatization Reduces remaining words to canonical base lemmas.
Aarya Yugant Mhatre

Stage 02: Feature Engineering Workflow

N-Grams Extracts sequential groupings of n continuous words.
→
Term Frequency (TF) Tracks the raw density of terms in a document.
→
Inverse Document Freq (IDF) Evaluates word rarity across the broader corpus.
→
TF-IDF Matrix Combines scores to weight contextually unique words.
Shraddha Sandip Salvi

Stage 03: Text-to-Vector Workflow

Word2Vec Builds static, shallow continuous neural embeddings.
→
FastText Generates subword embeddings to handle OOV tokens.
→
Embeddings Maps semantic features into compact dense vector clusters.
→
Transformer Models Utilizes dynamic attention for context-aware states.
